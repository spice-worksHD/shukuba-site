SHUKUBA 管理画面
操作マニュアル

このページは管理画面の使い方を説明する手順書です。機能が追加・変更されるたびに更新します。

対象読者
SHUKUBA（滋賀県古民家民泊）の宿泊管理を担当するオーナー・スタッフ向けです。

画面の全体構成

管理画面は上部のタブで5つのセクションに分かれています。

タブ名主な用途
予約履歴・サマリー今日の到着・出発状況の確認、予約一覧の操作（取消・LINE・チェックイン・チェックアウト）
宿泊台帳旅館業法対応の台帳確認、印刷プレビュー、リマインダー送信
料金・在庫カレンダー日別の空室管理、料金設定、本番サイトへの反映
決済設定クレジットカード・PayPayなど受付する決済方法の選択
🧪 テストデータ開発用テスト予約の作成・削除（本番公開前に削除予定）

ログイン・ログアウト

  1. 管理画面URL（https://shukuba-shiga.com/admin.html）を開く
  2. 管理パスワードを入力して「ログイン」を押す
  3. セッションはブラウザを閉じるまで保持されます（再起動後は再ログインが必要）
⚠ 注意
パスワードを他者に共有しないこと。パブリックなWi-Fiでの使用は避けてください。ログアウトは画面右上の「ログアウト」ボタンから。

サマリー数値の見方

「予約履歴・サマリー」タブ上部に6つの数値カードが表示されます。

カード内容
今日到着本日チェックインの予約件数と氏名
今日出発本日チェックアウトの件数
宿泊中現在滞在中のゲスト数
要確認
（赤枠で表示）		アクションが必要な予約。「未台帳・未コード送信あり」と表示される場合は、チェックインフォームの入力待ち または ドアコード未送信の状態
今月売上当月に作成された（キャンセル除く）予約の合計金額
今後の売上予定チェックイン日が未来の予約の合計金額

月次売上フィルター

「今月 / 来月 / 再来月」の売上ボタンをタップすると、その月のチェックイン予約だけに絞り込んで予約一覧を表示します。

  • ボタンをもう一度押すか「フィルター解除」で全件表示に戻る
  • 金額はキャンセル済みを除いた合計

絞り込み検索

フィルター説明
氏名・メール検索文字を入力すると即座に絞り込み
全施設物件で絞り込み
全状態予定（未チェックイン）・完了・キャンセルで絞り込み
日付（〜）チェックイン日の範囲で絞り込み
クリアすべてのフィルターを解除

「状況」列のバッジについて

各予約行の「状況」列に以下のバッジが表示されます。

バッジ意味
台帳済ゲストがチェックインフォームの記入を完了している
台帳未チェックインフォームがまだ未記入
LINE済ゲストがLINEを友だち追加・連携済み
LINE未LINE未連携（LINEボタンが表示されない）
📷 1/1（緑）外国籍ゲストのパスポート写真が必要枚数揃っている
📷 0/1（赤）パスポート写真が不足（ドアコード送信がブロックされる）
鍵送信済ドアロック番号をLINEで送信済み
退出済チェックアウト処理完了

「操作」列のボタンについて

ボタンできること・条件
取消 予約をキャンセルにする。日程が再予約可能になる。履歴には「キャンセル」として残る。
LINE LINE連携済みのゲストにテストメッセージを送る（LINE済バッジがある場合のみ表示）
📷確認 アップロードされたパスポート写真を新しいタブで確認できる（写真がある場合のみ表示）
🔑送信 パスポート確認後にドアロック番号をLINEで送信する。
条件: 台帳済 ＋ LINE済 ＋ ドアコード未送信。
外国籍ゲストがいる場合はパスポート写真が揃っていないとブロックされる。
退出確認 チェックアウト日が来た後に押す。LINE連携済みの場合、サンクスメッセージも同時送信。
✉サンクス 退出済み後に手動でサンクスメッセージだけ送る（未送信の場合のみ表示）

CSVダウンロード

予約一覧右上の CSVダウンロード を押すと、現在の絞り込み条件に関係なく全予約データをCSVで保存できます。Excelで開いて売上管理などに活用できます。

ヒント
CSVにはBOM（文字化け防止）付きでエクスポートするため、Excelで開いても文字化けしません。

台帳一覧の見方

「宿泊台帳」タブの上部に台帳の記入状況サマリー（本日・明日のチェックイン件数、記入済み数）が表示されます。

フィルターボタン

ボタン内容
すべて全予約を表示
未記入チェックインフォームをまだ入力していないゲストのみ
記入済みフォーム入力完了のゲストのみ

記入済みの行には「印刷」ボタンが表示されます。

印刷プレビュー

  1. 宿泊台帳一覧で「記入済み」の行の「印刷」ボタンを押す
  2. 新しいタブが開き、宿泊者名簿が表示される（自動で印刷ダイアログも開く）
  3. 内容を確認し、プリンターで印刷する（旅館業法上の保管用）

印刷プレビューの内容

  • 宿泊期間・物件名
  • 代表者情報（氏名・フリガナ・住所・電話番号・職業・国籍・旅券番号・到着予定）
  • パスポート写真（外国籍ゲストのみ）
  • 同行者情報（氏名・国籍・旅券番号・住所・備考）
  • 記入完了日時
旅館業法対応
旅館業法第6条に基づき、外国人宿泊者については旅券番号・国籍の記録が義務付けられています。印刷した名簿は3年間保管してください。

チェックインリマインダー送信

明日チェックインのゲストに、チェックインフォームのURLをメール/LINEで送ります。

内容
自動送信毎日09:00（JST）に自動実行。未送信のゲストのみが対象。
手動送信「明日チェックインの方へ今すぐ送信」ボタンで即時送信。二重送信にはならない（送信済みはスキップ）。
注意
LINE連携していないゲストにはメール送信のみになります。LINE連携は予約時にLINE公式アカウントを友だち追加することで連携されます。

自動チェックアウト手動実行

チェックアウト日が来た（または過ぎた）予約を一括で「退出済み」にします。

内容
自動実行毎日12:00（JST）に自動実行
手動実行「今すぐ退出完了」ボタンで即時処理。LINE連携済みゲストにはサンクスメッセージも同時送信。

メッセージテンプレート

台帳タブの下部にアコーディオン形式で2種類のテンプレートがあります。ヘッダーをタップして展開してください。

① LINEリマインダー本文テンプレート

チェックイン前日に自動送信するLINEメッセージの文面を編集できます。

変数置き換えられる内容
{checkinUrl}チェックインフォームのURL
{roomName}物件名
{checkin}チェックイン日
{name}予約者氏名

② チェックアウト後サンクスメッセージ

退出確認時または「✉サンクス」ボタン押下時にLINEで送信する感謝メッセージを編集できます。

変数置き換えられる内容
{name}予約者氏名
{roomName}物件名
{checkout}チェックアウト日
編集後は「保存」を忘れずに
テンプレートを変更したら必ず「保存」ボタンを押してください。押さずにタブを閉じると変更は失われます。

カレンダーの操作

「料金・在庫カレンダー」タブでは21日分の在庫・料金を管理できます。

操作内容
← 前の週 / 次の週 →表示期間を7日ずつ移動
今日本日を起点に戻す
横スクロール日付列が多い場合はカレンダーを横にスクロールして確認

売止・販売中の切り替え

カレンダーの「状態」行のセルをタップ・クリックして切り替えます。

表示意味操作
販売中予約受付可能な状態クリックすると「売止」に切り替わる
売止予約ブロック中（予約できない）クリックすると「販売中」に戻る
予約済実際の予約が入っている日クリックできない。取消は「予約履歴」タブから

日別特別料金の設定

カレンダーの「料金」行の入力欄に数値を入力すると、その日だけの特別料金に設定できます。

  • 空欄にすると基本料金に戻ります
  • 入力欄のプレースホルダー数値が基本料金です
  • 繁忙期・連休など料金を上げたい日に使います
  • 設定後は必ず「本番サイトに反映する」を押すこと

基本料金・定員設定

各物件の基本的な料金・人数設定です。

PCの場合

カレンダー左端の物件名列（スクロールしても固定されている列）内に入力欄があります。

スマホの場合

カレンダーの下にスクロールすると「部屋の基本料金・定員設定」セクションが表示されます。

項目内容
基本料金（円）1泊あたりのデフォルト料金
最小人数予約受付する最少人数
最大人数予約受付する最大人数
超過1名あたり追加料金基本人数を超えた場合の1名あたり追加料金
注意
基本料金・定員を変更した後も、「本番サイトに反映する」を押さないとお客様の予約画面には反映されません。

本番サイトへの反映

  1. 料金・在庫カレンダーで設定を変更する（売止・料金など）
  2. 画面右上（またはカレンダー上部）のステータスを確認。「⚠ 未反映の変更があります」と表示されたら反映が必要
  3. 本番サイトに反映する ボタンを押す
  4. 「✓ 本番サイトの内容は最新です」に変わったら完了
ヒント
在庫の「売止・販売中」切り替えは即座に保存されますが、料金変更と同様に「本番サイトに反映する」が必要です。ゲストの予約画面に反映させるために必ず押してください。

決済方法の設定

「決済設定」タブで、予約画面に表示する支払い方法を選択できます。

決済方法説明
クレジットカード（事前決済）Stripeによるオンライン決済
PayPay（事前決済）PayPayオンライン決済

チェックを入れた方法のみ予約フォームに表示されます。「保存」を押して反映してください。

チェックインの流れ（全体）

① ゲストが予約
→
② LINE公式アカウントを友だち追加
→
③ 前日09:00にリマインダー自動送信
→
④ ゲストがチェックインフォーム記入
→
⑤ 管理者が台帳・パスポートを確認
→
⑥ 🔑送信ボタンでドアコードをLINE送信
外国籍ゲストの場合
④のフォーム記入時にパスポート写真をアップロードします。管理者は⑤で「📷確認」ボタンから写真を確認し、問題なければ⑥へ進んでください。写真が不足している場合、🔑送信はブロックされます。

チェックアウトの流れ

① チェックアウト日の12:00に自動処理
→
② LINE連携済みゲストにサンクス自動送信
→
③ 管理画面で「退出済」バッジを確認

手動で対応する場合

  • 予約一覧の「退出確認」ボタンを押すと即時退出処理＋サンクス送信
  • サンクスメッセージだけ後から送りたい場合は「✉サンクス」ボタン
